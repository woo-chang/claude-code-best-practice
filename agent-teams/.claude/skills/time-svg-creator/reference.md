
### 플레이스홀더

| 플레이스홀더 | 교체 값 | 예시 |
|-------------|-------------|---------|
| `{{TIME}}` | 입력의 `time` 값 | `14:30:45` |
| `{{TIMEZONE}}` | 입력의 `timezone` 값 | `GST (UTC+4)` |
| `{{DATE}}` | `formatted`에서 추출한 날짜 (처음 10자) | `2026-03-12` |
| `{{FORMATTED}}` | 전체 `formatted` 문자열 (마크다운에서만 사용) | `2026-03-12 14:30:45 +04` |

### 디자인 명세

| 속성 | 값 |
|----------|-------|
| 크기 | 400 x 250 px |
| 모서리 반경 | 16 px |
| 배경 | 선형 그라디언트: `#0a1628` → `#1a2744` (짙은 남색에서 짙은 파란색) |
| 제목 | `#8892b0` (흐린 파란색), 16px 세미볼드 |
| 시간 표시 | `#ffffff` (흰색), 52px 볼드 |
| 타임존 | `#64ffda` (청록색 강조), 16px |
| 위치 | `#ccd6f6` (연한 파란색), 14px |
| 날짜 | `#8892b0` (흐린 파란색), 12px |
| 폰트 | `sans-serif` |
| 모든 텍스트 | 가운데 정렬 (x=200에서 `text-anchor="middle"`) |
